# resources.md 格式

`resources.md` 是该主题精选过的可信来源集合。讲解所用的知识应当从这里取材，而不是依赖参数化的猜测。

## 结构

```md
# {主题} 资源

## 知识

- [书籍：《力量训练的科学与实践》—— Zatsiorsky & Kraemer](https://example.com)
  关于训练编排与适应的奠基性教材。适用：周期化、恢复、强度区间相关的一切。
- [文章："How Much Should I Train?" —— Greg Nuckols（Stronger By Science）](https://example.com)
  对训练量界标的循证综述。适用：每个肌群每周的目标组数。
```

## 规则

- **只收高可信度来源。** 优先选择一手资料、公认专家、同行评议作品。如果一份资源是包装成"教育"的营销，就不要收。
- **每一条都加注解。** 三个月后看，光秃秃的链接毫无价值。加一行：它讲的是什么，以及什么时候会用到它。
- **明确呈现缺口。** 如果某个使命所需的领域没有好的资源，写一节 `## 缺口` 来列出缺失的内容。这能驱动未来的搜索。
- **无情地修剪。** 一份后来被证明是错误的、肤浅的，或偏离使命的资源，应当被删除，而不是被埋起来。**五个锋利的来源胜过三十个平庸的来源。**
