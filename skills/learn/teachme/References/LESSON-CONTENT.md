# 课程内容（LESSON-CONTENT.md）

## .md 写什么

骨架（伪代码，1/2/6/7 必有）：

```
1. # 标题
2. > 一句话点出本课的"小胜利"，并连接到使命（mission.md）
3. ## 知识      —— 紧凑要点；每个主张挂引用 [^src]
4. ## 练习      —— 技能型课程必有；含 quiz 或步骤清单
5. ## 反馈      —— 答案/复盘
6. ## 主要来源  —— 一条最高可信的外部资源链接
7. ## 接着问我  —— 提醒用户向 agent 追问
8. 末尾：脚注式引用 [^src]: ...
```

测验题选项**等字数**（最好等字符数），不通过排版给暗示。
